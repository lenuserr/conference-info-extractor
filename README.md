# Conference Info Extractor

## 📌 Описание проекта

Проект посвящён извлечению структурированной информации с сайтов научных конференций (например, CFP-страниц) и сохранению её в базу данных для последующего анализа.

Цель — собрать единый датасет конференций по всему миру и на его основе построить систему оценки качества конференций (в том числе с использованием методов машинного обучения).

---

## 🎯 Цели

* Извлекать структурированные данные по URL конференции
* Нормализовывать и сохранять данные в БД
* Анализировать конференции по различным параметрам
* В будущем — построить модель, автоматически оценивающую уровень конференции

---

## 🔍 Извлекаемые данные

Система парсит сайты конференций и извлекает следующую информацию:

### 🧾 Общая информация

* Название конференции
* Аббревиатура
* Год проведения

### 📅 Даты

* Даты проведения
* Дедлайн подачи статей (submission deadline)
* Дата уведомления (notification)
* Camera-ready deadline

### 📍 Место проведения

* Город
* Страна
* Формат (offline / online / hybrid)

### 🧠 Тематика

* Основные темы (topics, areas of interest)
* Ключевые слова

### 👥 Организация

* General chairs
* Program committee
* Аффилиации участников

### 🎤 Контент

* Keynote-спикеры
* Workshops
* Tutorials

### 📚 Публикации

* Издатель (IEEE, ACM, Springer и др.)
* Индексация (Scopus, Web of Science, DBLP)

### 🔗 Ссылки

* Официальный сайт
* CFP-страница
* Система подачи (EasyChair, OpenReview и др.)

---

## 🧠 Зачем это нужно

Научные конференции сильно различаются по уровню и качеству.
Данный проект позволяет собрать структурированные признаки, которые можно использовать для оценки:

* престижности конференции
* научного уровня
* вероятности того, что конференция является "хищной" (predatory)

Примеры сигналов качества:

* наличие авторитетного издателя (ACM, IEEE)
* индексация в Scopus / WoS
* состав организационного комитета
* адекватность дедлайнов
* наличие keynote-спикеров
* регулярность проведения конференции

## 🌐 Источники данных о конференциях

* [WikiCFP](http://www.wikicfp.com/cfp/) — агрегатор Call for Papers
* [Conference Alerts](https://conferencealerts.com/) — каталог конференций по всему миру

* [WASET](https://waset.org/conferences) — база международных конференций
